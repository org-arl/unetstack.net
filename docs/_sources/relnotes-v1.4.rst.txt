UnetStack and simulator v1.4 release notes
==========================================

May 26, 2018

New features
------------

* Moved open source contributions to `unet-contrib <https://github.com/org-arl/unet-contrib>`_.
* `Python API <https://github.com/org-arl/unet-contrib/tree/master/contrib/Unet-Python-API>`_ for UnetStack access (beta).
* `C API <https://github.com/org-arl/unet-contrib/tree/master/contrib/Unet-C-API>`_ for UnetStack access (beta).
* New `Scheduler <svc-50-scheduler.html>`_ service and agent.
* New web interface framework.
