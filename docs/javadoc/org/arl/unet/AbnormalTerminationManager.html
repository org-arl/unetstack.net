[Java] Class AbnormalTerminationManager

        • org.arl.unet.AbnormalTerminationManager

        • Agent to handle abnormal termination of agents and perform appropriate action. Handles notifications sent to topic Topics.ABNORMAL_TERMINATION.

          See Also:
          UnetAgent
