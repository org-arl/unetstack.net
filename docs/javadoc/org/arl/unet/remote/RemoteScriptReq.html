Package: org.arl.unet.remote

[Java] Class RemoteScriptReq

      • org.arl.unet.remote.RemoteScriptReq

      • Request message to execute a groovy script in a remote node. Code example:

            /// executes hello.groovy script placed in scripts folder on node 2.
    RemoteScriptReq req = new RemoteScriptReq();
    req.setAckReq(true);
    req.setTo(2);
    req.setRecipient(agent.agentForService(Services.REMOTE));
    req.setScriptName("hello");
    Message rsp = agent.request(req, timeout);
