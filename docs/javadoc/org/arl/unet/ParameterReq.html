Package: org.arl.unet

[Java] Class ParameterReq

      • org.arl.unet.ParameterReq

      • Request one or more parameters of an agent. The response can be retrieved by ParameterRsp.
        Code example:

           ParameterReq req = new ParameterReq(agent.agentForService(Services.PHYSICAL));
   req.get(PhysicalParam.propagationSpeed);
   req.get(PhysicalParam.rxEnable);
   ParameterRsp rsp = (ParameterRsp) agent.request(req, 1000);
