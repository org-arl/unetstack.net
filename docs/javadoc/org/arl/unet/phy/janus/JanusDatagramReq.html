Package: org.arl.unet.phy.janus

[Java] Class JanusDatagramReq

        • org.arl.unet.phy.janus.JanusDatagramReq

        • Janus Datagram request message to send across data to a destination. Code example:

             byte[] data = {1,2,3,4};
   JanusDatagramReq req = new JanusDatagramReq(agent.agentForService(Services.PHYSICAL));
   req.setTo(Address.BROADCAST);
   req.setJanusVersion(2);
   req.setUserClass(5);
   req.setType(Physical.DATA);
   req.setScheduled(true);
   req.setData(data);
   agent.send(req);
