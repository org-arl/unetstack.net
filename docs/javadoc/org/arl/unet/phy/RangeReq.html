Package: org.arl.unet.phy

[Java] Class RangeReq

      • org.arl.unet.phy.RangeReq

      • Request message to fetch range and sync information from a destination.

        Code example: 

         //Sync request
   RangeReq req = new RangeReq();
   req.setTo(2);
   req.setRecipient(agent.agentForService(Services.RANGING));
   org.arl.fjage.Message rsp = agent.request(req, 1000);
 //range notification will display the range information
