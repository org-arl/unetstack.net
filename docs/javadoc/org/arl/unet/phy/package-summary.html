Package org.arl.unet.phy Description

Physical layer interface definition.
  • Class Summary 
    Class Description
    BadFrameNtf Notification message for reception of corrupted or bad physical frame.
    BadRangeNtf Notification message of bad ranging information, when the range is out of bounds.
    BeaconReq Request message to transmit a beacon.
    ClearSyncReq Request message to clear the sync info.
    CollisionNtf Notification message for collision of frames.
    PhyTickerBehavior A behavior that is executed every specified period based on the physical layer clock.
    Physical Physical layer constants.
    RangeNtf Notification message for providing ranging information such as from, range (meters) and clock offset (us).
    RangeReq Request message to fetch range and sync information from a destination.
    Ranging Agent providing ranging functionality.
    RxFrameNtf Notification message for received frame.
    RxFrameStartNtf Clock Notification message for start of reception.
    SyncInfoReq Request message to fetch sync-info for a given node.
    SyncInfoRsp Response message for SyncInfoReq request message.
    TxFrameNtf Notification of a successfully transmitted frame with tx timestamp.
    TxFrameReq Request message to transmit a frame using physical agent.
    TxRawFrameReq Raw frame transmission request.
