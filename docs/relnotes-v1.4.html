    UnetStack and simulator v1.4 release notes¶

    May 26, 2018

    New features¶

    • Moved open source contributions to unet-contrib.
    • Python API for UnetStack access (beta).
    • C API for UnetStack access (beta).
    • New Scheduler service and agent.
    • New web interface framework.

    Changes¶

    • Improved documentation.
    • Switched to JDK 1.8, Groovy 2.4.15 and fjåge 1.4.2.
    • Extended default address space to 8 bits (255 nodes).
    • Several minor API enhancements.

    Bugfixes¶

    • Several minor bug fixes.
