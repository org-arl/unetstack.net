    Route Maintenance¶

    Service¶

    ROUTE_MAINTENANCE – Route maintenance service

    Agents offering the Route Maintenance service generate route discovery/change notifications to allow routing agents to maintain routing tables. They support a set of messages that are explained below.

    Implementations¶

    • RouteDiscoveryProtocol – provides route discovery and maintenance service by flooding

    Requests and Responses¶

    Request Possible Responses Description
    RouteDiscoveryReq AGREE, REFUSE, FAILURE Request for route discovery to specified node
    RouteTraceReq AGREE, REFUSE, FAILURE Request for trace current route to specified node

    Notifications¶

    Notification Topic Description
    RouteDiscoveryNtf default Notification that a route has been discovered
    RouteTraceNtf default Notification with a route trace to a node
